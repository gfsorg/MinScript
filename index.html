MinScript Documentation

Welcome to the official documentation for MinScript—the world's most lightweight, non-bloated scripting language. Designed to do exactly two things perfectly: manage variables and print data.

Getting Started

To run MinScript files (.ms), you need to compile the msc.c compiler engine using a standard C compiler.

1. Compilation

gcc msc.c -o msc

2. Execution

Pass your script straight to the executable binary:

./msc script.ms

Variable Declaration

MinScript supports dynamically-typed variables initialized via the let keyword. Variables can dynamically accept integers, string literals, or reference other pre-existing variables.

# Declaring an Integer
let score = 100

# Declaring a String
let message = "Hello, World!"

# Copying a variable reference
let backupScore = score

Printing

The print command outputs content directly to standard output (stdout). It handles raw strings, integers, and variables flawlessly.

print "This is a literal string"
print 42
print message

Core Philosophy

MinScript was built on the foundation that software has strayed too far from simple constraints. If you ever need to reaffirm this absolute objective truth directly within your codebase, you can invoke the primary compiler statement:

every other language has too much bloat

Output response from compiler:

yup, correct!
